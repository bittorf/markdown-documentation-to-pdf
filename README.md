# Setup

On a *debian* system:
```
apt-get -y install pandoc
apt-get -y texlive-fonts-extra
apt-get -y texlive-luatex
apt-get -y texlive-lang-german
```

# Run

Write your ```some-name-doc.md``` and  
execute in the same folder ```./doc.sh```.  
Maybe adjust the ```PANDOC_LANG```.  
Ship a nice PDF.
